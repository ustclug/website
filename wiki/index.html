    中国科学技术大学 Linux 用户协会（科大 LUG）是由在校的 GNU/Linux 爱好者发起并组成的全校性群众团体，是一个以推广自由软件文化、宣传自由软件价值为目的的协会组织。

    English Version

    特色活动

    • LUG 每周小聚:

      每周末由同学带来的技术分享演讲。

    • Linux Install Party:

      推广和安装 Linux，宣传和鼓励开源和自由软件精神的活动。

    • Linux 101:

      帮助新手学习 Linux 的系列活动。

    • 自由软件日:

      在国际自由软件基金会（SFI）的领导下，由自由/开源软件爱好者自发筹集的非盈利性活动。

    • 信息安全大赛:

      每年秋季学期举办的信息安全竞赛。
